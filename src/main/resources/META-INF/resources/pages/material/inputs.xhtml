Material inputs This page shows how to apply style to input components.

Material design inputs must be surrounded by a div with class ui-material and its label must be placed after the input.

Note that p:calendar needs to call a javascript on date select event, that is needed because Primefaces doesn't trigger any javascript event on date select and we use javascript (adminMaterial function) to make the label float when element value is not empty.

This will not be needed when this issue is fixed.

Material inputs depend on Admin Template javascript to work, so if you use admin-theme without template you must add admintemplate.js in you application
